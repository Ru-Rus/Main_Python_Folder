## Acronym Generator CLI

Turn any phrase into an instant acronym. Clean. Fast. Simple.

# What This Program Does

• Prompts you to enter a phrase
• Extracts the first letter of each word
• Converts letters to uppercase
• Displays the generated acronym
• Keeps running until you stop it

## Example
# Input
    random access memory
# Output
    Acronym: RAM

# How to Run

Step 1
Install Python 3

Step 2
Save the file as:
acronym_generator.py

Step 3
Open your terminal or command prompt

Step 4
Run:
acronym_generator.py

# How to Stop

• Windows → Press Ctrl + Z then Enter
• Mac/Linux → Press Ctrl + D

# Requirements

• Python 3.x
